 GET /api/v1/state-data

Implementation Notes

Get the list of states with daily reports.

Response Class (Status 200)

            
                [
                    {{ '{' }}
                      "name" : "",
                      "confirmed" : "",
                      "active" : "",
                      "recovered" : "",
                      "deaths" : "",
                      "dayReports" : [
                                        {{ '{' }}
                                            "date" : "",
                                            "confirmed" : "",
                                            "active" : "",
                                            "recovered" : "",
                                            "deaths" : ""
                                        {{ '}' }},
                                        {{ '{' }}
                                            ...
                                        {{ '}' }}
                                      ]
                      "districts" : [
                                        {{ '{' }}
                                            "name" : "",
                                            "confirmed" : "",
                                            "active" : "",
                                            "recovered" : "",
                                            "deaths" : ""
                                        {{ '}' }},
                                        {{ '{' }}
                                            ...
                                        {{ '}' }}
                                    ]
                     {{ '}' }},
                     {{ '{'}}
                        ...
                     {{ '}'}}
                ]

Response Content Type : application/json

Parameters

         1) Name : districts
            Type : boolean
            Values : true - > Includes list of districts in the response.
                     false - > Response will not contain the data for districts.
            
    
         2) Name : dayReports
            Type : boolean
            Values : true - > Includes daily reports for the states.
                     false - > Response will not contain daily reports for the states.

Sample Request

http://covidmapper.in/api/v1/state-data?districts=true&dayReports=false
